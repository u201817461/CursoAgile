# Simulador de Velocidad Web

Este proyecto es un **simulador interactivo de velocidad** desarrollado en HTML, CSS y JavaScript puro. Permite calcular la velocidad a partir de la distancia y el tiempo, y visualizar cómo las variables afectan el resultado mediante gráficos interactivos.

---

## Funcionalidades

- **Cálculo de velocidad:** v = d / t, donde `d` es distancia y `t` es tiempo.
- **Campos de entrada:** para distancia y tiempo.
- **Validación de datos:** no se permiten valores negativos o no numéricos.
- **Visualización de resultados:** muestra la velocidad calculada en pantalla.
- **Gráficos interactivos:** historial de los últimos 10 cálculos.
- **Actualización dinámica:** permite modificar las variables y recalcular sin recargar la página.
- **Interfaz simple y clara:** diseñada para uso educativo y demostrativo.
- **100% Front-end:** no requiere backend.
